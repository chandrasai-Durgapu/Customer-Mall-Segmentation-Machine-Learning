# Customer-Mall-Segmentation-Machine-Learning
Customer Mall Segmentation using Unsupervised Learning like K-Means machine learning and MLFlow
fastapi is pending

## Clone the repository
```bash
git clone https://github.com/chandrasai-Durgapu/Customer-Mall-Segmentation-Machine-Learning.git
```
```bash
cd Customer-Mall-Segmentation-Machine-Learning
```

## Create Virtual Environment
```bash
python -m venv segment-ml-02
```

## Activate Virtual Environment
```bash
segment-ml-02\Scripts\activate
```

## Install Dependencies
```bash
pip install -r requirements.txt
```
